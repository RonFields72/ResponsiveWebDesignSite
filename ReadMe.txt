In order to run this application you need to have the following software installed on your machine:
Telerik UI for ASP.NET AJAX Q1 2014 (2014.1.225)
Microsoft SQL Server Express 2008
Microsoft .NET framework 4.0

The database is located in the App_Data folder. You may need to modify your web.config and 
app.config's connectionStrings, if necessary, to match your instance name and database location.
